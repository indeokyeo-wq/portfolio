01 방향성

매출 확장의 상위 방향성. 솔루션 6 카테고리와 세부 액션은 관리·효율·확장에서 파생.

관리
기존 IP 매출 증대

보유 트래픽·IP·광고 인벤토리에서 외부 유출 매출 회수. 단가를 산업 평균 수준까지 상향. 외부 자원 없이 가장 빠른 손익 개선.

전략적 근거
  • JustPlay 자체 채널 누적 인스톨 2,500만+ 미활용
  • 외부 광고 SDK 수수료 10~30% 누수 추정
  • 3사 합산 DAU 360만 분산 운영, 시너지 미발생
기대 효과 : 관리 체계 구축으로 광고 매출 +25~50%, Springcomes 매출 2~5배 여지 (단가 상향 가정)
→ 세부 액션은 솔루션 탭 관리 카테고리에서 확인
효율
원가 효율화·양산 가속

Lihuhu 양산 라인 + 인건비 효율 + 검증 IP의 양산화로 동일 산출 대비 비용 절감, 신작 출시 페이스 가속.

전략적 근거
  • Lihuhu 영업이익률 ~25%, 검증된 양산 시스템
  • 베트남 IT 인건비 한국 대비 30~35% 수준
  • Springcomes 신작 연 4~5종, 양산 캐파 부족이 병목
기대 효과 : 신작 페이스 4~5종 → 8~12종/년, 게임당 운영 리소스 -30%
→ 세부 액션은 솔루션 탭 효율 카테고리에서 확인
확장
권역 확장 + 신규 M&A

2027년 일본 권역 확장으로 시작, 2028년 누적 수익으로 신규 M&A(캐주얼·SLG 등 인접 장르) 본격화.

전략적 근거
  • 3사 매출 70~80% 북미·유럽 집중, 권역 편중
  • 글로벌 캐주얼 시장 점유율 0.6%로 확장 여지 큼
  • Springcomes 일본 친화 머지 IP 4종 보유
기대 효과 : 일본 머지 월 ARPU 4,130원, M&A IP 매출 기여 +200~800억
→ 세부 액션은 솔루션 탭 확장 카테고리에서 확인
