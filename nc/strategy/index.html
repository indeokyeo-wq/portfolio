01 방향성

매출 확장의 상위 방향성. 솔루션 탭의 6대 카테고리와 세부 액션은 모두 관리·효율·확장 3가지 방향에서 파생.

관리
기존 IP 매출 증대

보유 트래픽·IP·광고 인벤토리에서 외부 유출 매출을 회수, 단가를 산업 평균 수준까지 정상화. 외부 자원 투입 없이 가장 빠른 손익 개선 영역.

전략적 근거
  • Springcomes DAU 250만 / ARPDAU ~30원 — 광고형 BM 정상치 vs 단가 누락 분기 진단
  • 외부 광고 SDK 의존 — 수수료·중간 마진 누수 추정 (mediation 표준 10~15%, 리워드 결합 시 30%대까지 관측)
  • 3사 합산 DAU 360만 트래픽이 분산 운영, 시너지 미발생
  • JustPlay 누적 인스톨 2,500만+의 자체 채널 자산 미활용
핵심 액션 (2026~27)
  • 외부 광고 수수료 내재화 — JustPlay 인벤토리의 3사 통합 허브화 검토
  • 광고 운영 표준화 — Header Bidding·Floor Price·mediation 통합 도입
  • 3사 게임 교차 추천 확대 — 자체 트래픽 활용도 상향
정량 효과 — 단가 누락 검증 시 Springcomes 매출 2~5배 여지, 광고 매출 +25~50% (단가·점유율 정상화 가정)
효율
원가 효율화·양산 가속

Lihuhu 양산 라인 + 인건비 효율 + 검증 IP의 양산화로 동일 산출 대비 비용 절감, 신작 출시 페이스 가속.

전략적 근거
  • Lihuhu 영업이익률 ~25% (Voodoo·Homa 벤치마크) — 검증된 양산 시스템
  • Lihuhu 100+ 게임 양산 라인 가동 중 — 즉시 활용 가능 자원
  • 베트남 IT 인건비 한국의 30~35% 수준 — 동일 산출 대비 인건비 효율
  • Springcomes 신작 연 4~5종 — 양산 캐파 부족이 매출 다양화 병목
핵심 액션 (2027~)
  • Lihuhu 개발 리소스 Springcomes 이식 — 한국 기획 + 베트남 양산 3개월 사이클
  • 검증 IP 양산화 극대화 — 변주·시리즈 신작
  • AI 운영 자동화로 170+ 게임 리소스 절감
정량 효과 — 신작 페이스 4~5종 → 8~12종/년, 게임당 운영 리소스 -30%
확장
권역 확장 + 신규 M&A

전략적 근거

  • 글로벌 캐주얼 시장 점유율 0.6% — 99% 미진입 영역
  • Springcomes 일본 친화 머지 IP 4종 보유 (머지 페코·헬로타운 등)
  • 26~27년 관리·효율 누적 매출로 M&A 재투자 여력 형성 가능

01 관리

보유 트래픽·IP·광고 인벤토리에서 외부 유출 매출을 회수하고 단가를 산업 평균까지 정상화. Phase 1~2의 가장 빠른 손익 개선 가능 영역 (실측 검증 후 우선순위 확정).

1

1.1
JustPlay 인벤토리를 3사 통합 광고/마케팅 허브로

현재 (추정): Lihuhu·Springcomes는 외부 광고회사(AppLovin·Unity·AdMob) 의존도 높음. 광고비 일부가 외부 수수료(10~30% 추정)로 유출.

액션: Lihuhu·Springcomes의 광고 노출을 JustPlay 자체 인벤토리로 일부 라우팅. 동시에 JustPlay 메인 앱에서 Lihuhu·Springcomes 게임 추천(cross-promotion). mediation 수수료 일부 내재화 + 신규 인스톨 자체 채널 확보.

임팩트 (가설 검증 시): 광고 마진 +10~25% / 합산 매출 1,480억 기준 +150~370억 범위 추정. 실제 효과는 mediation 계약 구조·단가 갭에 따라 변동.

2
마케팅 (UA). 더 효율적으로, 더 싸게
현재 UA 채널: Meta · Google · TikTok · 외부 SDK 분산. 자체 채널 활용 부족
2.1
JustPlay 메인 앱에서 Springcomes·Lihuhu 게임 추천 (Cross-promo)

관리 1과 차이점: 같은 JustPlay 인벤토리지만 용도가 다름. 관리 1 = 외부 광고주 매출 측면 (광고주 → 우리 인벤토리 → SDK 수수료 회수). 여기 2.1 = 우리 UA 비용 측면 (우리 메인 앱에서 우리 게임 추천 → 외부 UA 비용 절감).

액션: JustPlay 메인 앱(누적 2,500만+, 매월 신규 50만 — App Store/Sensor Tower 추정)에서 Lihuhu·Springcomes 게임 자동 추천. 자체 트래픽이라 외부 광고비 0원. Cross-promo 비율 첫해 10% → 3년차 40% 목표.

임팩트: Cross-promo 비중에 비례한 CAC 절감 효과 (자체 채널 CPI ≈ 외부 1/3~1/5 가정 시 가중평균 CAC -15~30%) / 신규 인스톨 +20~40%

2.2
인플루언서·UGC 마케팅 — Royal Match 모델 이식

현재: 페이드 광고 위주. 유기적 입소문 마케팅 부족.

액션: 머지 게임의 "진행 영상이 만족감 있는 콘텐츠" 특성을 활용 → TikTok·Shorts·YouTube Shorts 자생적 확산 유도. 인플루언서 50명 시드 + UGC 캠페인.

임팩트: 인플루언서 CPI ~$1.0~1.5 vs Meta 페이드 UA $2~3 → CAC 30~50% 절감.

Royal Match 사례
Dream Games(터키)가 2021 출시한 매치-3 게임. TikTok·인스타·유튜브 인플루언서 활용으로 페이드 UA 의존도 줄이고 유기적 확산 극대화. 2024년 글로벌 매출 $2B+ 추정 (Sensor Tower·AppMagic). 동일 장르 머지 게임에 적용 가능한 검증된 모델.

※ 산업 데이터: TikTok 인플루언서 마케팅 평균 CPI는 페이드 UA의 30~60% (HypeAuditor·Influencer Marketing Hub 2025).

2.3
다채널 동시 가동 → 효율 비교 → 자원 재배분

현재 (추정): 3사가 각자 UA 운영 → 3사 데이터 통합·비교 체계 도입 여부 30일 진단 필요.

액션: 주요 자동 입찰 채널(Meta Advantage+ · Google App Campaign · TikTok · AppLovin AXON) 도입 여부 진단 → 통합 회수율 모델 미구축 시 3사 학습 데이터 풀링으로 알고리즘 학습 가속. 채널별 ROAS 비교 → 효율 좋은 채널에 자원 재배분.

임팩트: 채널별 효율 격차 시각화 → 광고 효율 +10~20%

용어 — 자동 입찰 알고리즘
Meta Advantage+: Meta(페이스북·인스타) AI가 자동 입찰·타겟팅. Google App Campaign: 구글이 자동으로 검색·디스플레이·유튜브 등 노출 최적화. AppLovin AXON: AppLovin(모바일 광고 1위 플랫폼)의 AI 자동 입찰 알고리즘. TikTok Smart+: TikTok 자동 입찰. 각자 강점 다름 → 동시 가동 후 효율 비교가 표준.
2.4
게임별 LTV 측정 → CAC 임계선 → 광고비 자원 배분 매트릭스

현재 (추정): "북미 70%" 같은 거시 지표만 외부 공개. 게임별·국가별·장르별 LTV/CAC 측정 체계가 통합 운영되고 있는지 30일 진단 1순위.

액션: 산업 best-practice 기준의 갭 진단 → 게임별·국가별·장르별 LTV(유저 1명 평생 매출) 측정 체계 강화 → CAC(획득 비용) 임계선 산출 → 광고 채널별 ROAS 매트릭스로 자원 배분 의사결정. 본인이 밀리에서 운영한 LTV 측정 체계(장르별 분리·CAC 임계선 규칙화) 직접 응용 가능.
