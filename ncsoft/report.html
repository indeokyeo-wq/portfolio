정량 측정 KPI 4축

3사 통합 운영의 성공·실패를 매주 정량으로 판정 가능한 의사결정 기준선입니다.

KPI ①
국가별 LTV/CAC 임계선

측정: 게임 × 국가별 유저 1명 평생 매출(LTV) ÷ 광고 획득비(CAC). 임계선: LTV/CAC ≥ 3.0 = 광고 확장 / 1.5~3.0 = 유지·최적화 / <1.5 = 채널 차단. 현 상태: 미공개·미측정 추정.

KPI ②
게임별 ARPDAU 추이

측정: 매일 게임 켜는 유저 1명당 매출(광고+IAP). 벤치마크: 매치-3 $0.10~0.15 / 머지 $0.08~0.12 / 하이퍼캐주얼 $0.03~0.05. 액션 트리거: ARPDAU가 벤치마크의 70% 이하 = 즉시 진단 대상.

KPI ③
인벤토리 Cross-promo 비율

측정: 4사 게임 광고 슬롯 중 다른 자회사 게임으로 전환된 유저 비율. 목표: 첫해 10% → 2년차 25% → 3년차 40%. 가치: 외부 UA 비용 0원으로 신규 인스톨 확보.

KPI ④
신작 D7 ROAS

측정: 신작 출시 7일 안에 광고비 회수율. 임계선: D7 ROAS ≥ 25% = 글로벌 확장 / 15~25% = 한정 시장 / <15% = 조기 종료. 벤치마크: Springcomes Hello Town D7 50% (산업 평균 이상).

통합 시너지의 정량 메커니즘 — JustPlay 2,366억(gross) vs Springcomes 280억(net) 매출 비대칭 해석

두 회사 매출의 회계 기준이 다릅니다.

  • JustPlay 2,366억 = 광고주가 직접 지급한 거래 총액(gross) 기준. 유저 보상이 약 50% 차감되어 영업이익률 11% 형성.
  • Springcomes 280억 = 외부 SDK 수수료 차감 후 받는 분배금(net) 기준. 광고주 실제 지출액은 약 2~3배 규모로 추정.
  • 핵심 메커니즘: Springcomes DAU 250만은 JustPlay(50만)의 5배. Springcomes 인벤토리를 JustPlay 채널로 라우팅 시 외부 수수료 절감 + gross 기준 매출 비선형 증가가 가능합니다. 엔씨가 발표한 통합 시너지의 정량 메커니즘입니다.
