STEP 1. 광고 배치 시점 발굴

목적: AI가 직접 게임을 플레이하며 광고 배치 최적 시점 발굴
  • ADB + Python으로 에뮬레이터 자동 플레이 및 스크린샷 캡처
  • Claude Vision API로 화면 인식 (광고 노출/실패/보상 화면)
  • JSON 로그 누적 및 분석 리포트 자동 생성 파이프라인
산출물: 게임별 광고 배치 시점 데이터 / 리워드 최적 유도 지점
⏱

STEP 2. 광고 피로도 감지

목적: AI가 광고 피로도 축적 패턴을 감지하여 교체 시점 알림
  • MMP API(Adjust/AppsFlyer) 연동하여 데이터 자동 수집
  • Python으로 채널별 CTR/ROAS/CPC 추이 자동 분석
  • 피로도 임계점 도달 시 Slack Webhook 자동 알림 발송
산출물: 소재별 상태 대시보드 (정상/주의/교체) / 최적 교체 주기
🔍

STEP 3. 인수 후보 게임 스크리닝

목적: 수동 시장 조사를 30분 AI 자동 분석으로 대체
  • Google Play / App Store 공개 지표 API 자동 수집
  • Claude API가 다차원 평가 (DAU/광고잠재력/리뷰 감성) 수행
  • 자체 알고리즘 기반 인수 적합도 점수 자동 스코어링
산출물: 후보 게임 순위 리포트 / 100점 만점 요약 리포트
